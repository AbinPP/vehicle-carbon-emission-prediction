# Vehicle Carbon Emission Prediction

## Project Overview
This project predicts the carbon emissions of vehicles using machine learning regression models.

The goal is to analyze vehicle characteristics such as engine size, fuel consumption, and cylinders to estimate CO2 emissions.

## Tools Used
Python  
Pandas  
Scikit-learn  
Matplotlib  
Seaborn  

## Machine Learning Models
Linear Regression  
Polynomial Regression  

## Key Features
- Data preprocessing
- Regression model training
- Model evaluation
- CO2 emission prediction

## Dataset
Vehicle dataset containing engine specifications and fuel consumption metrics used to estimate carbon emissions.
